# Histórias de Usuário - Sistema de Matrículas

## Épico 1: Acesso e Segurança
* [cite_start]**US01 - Autenticação:** Como um usuário do sistema, eu quero utilizar uma senha para validação do meu login, para que eu possa acessar a plataforma com segurança[cite: 19].

## Épico 2: Gestão Administrativa (Secretaria)
* [cite_start]**US02 - Manutenção de Dados:** Como secretaria, eu quero manter as informações sobre as disciplinas, professores e alunos, para garantir que os registros da universidade estejam corretos e atualizados[cite: 9].
* [cite_start]**US03 - Geração de Currículo:** Como secretaria, eu quero gerar o currículo para cada semestre, para que as disciplinas fiquem estruturadas e disponíveis para oferta[cite: 9].

## Épico 3: Portal do Aluno e Regras de Negócio
* [cite_start]**US04 - Efetuar Matrícula:** Como aluno, eu quero acessar o sistema durante o período de matrículas para me inscrever em 4 disciplinas de 1ª opção (obrigatórias) e em 2 alternativas (optativas)[cite: 11, 12].
* [cite_start]**US05 - Cancelar Matrícula:** Como aluno, eu quero acessar o sistema durante o período de matrículas para cancelar matrículas feitas anteriormente, permitindo o ajuste da minha grade curricular[cite: 12].
* [cite_start]**US06 - Capacidade Máxima:** Como sistema, eu quero encerrar as inscrições para uma disciplina quando ela atingir o limite de 60 alunos inscritos, para não exceder a capacidade permitida[cite: 14].
* [cite_start]**US07 - Viabilidade da Disciplina:** Como sistema, eu quero cancelar uma disciplina caso ela tenha menos de 3 alunos inscritos no final do período de matrículas, garantindo que apenas turmas viáveis fiquem ativas no semestre[cite: 13, 14].

## Épico 4: Integrações e Portal do Professor
* [cite_start]**US08 - Integração Financeira:** Como sistema de matrículas, eu quero notificar o sistema de cobranças logo após a inscrição do aluno no semestre, de modo que o aluno possa ser cobrado corretamente pelas disciplinas[cite: 15].
* [cite_start]**US09 - Consulta de Turmas:** Como professor, eu quero acessar o sistema para consultar quais são os alunos que estão matriculados em cada disciplina que ministro[cite: 18].
